Slipmat Labs

Experimenting and pushing the limits of Web technologies and DJ tools.

Tool Experiments (For DJs and Artists)

  • Slipmat Show Scripter - experimental tool for building radio show scripts (and tools for playing)
  • Slipmat EventPlanner - experimental tool for organizing events (like festivals) across multiple timezones
  • Kitten - metadata tool for querying Web site metadata and taking automated screenshots

APIs (For Developers)

Free to use APIs for all active Slipmat community members. We reserve the right to limit eccessive usage.

  • Discogs artist search - easy to use Discogs artist search API.
  • Spotify Playlist Parser - easy to use API for parsing Spotify playlists.
  • Tidal Playlist Parser - easy to use API for parsing Tidal playlists. (Note: this may not be working if Slipmat Tidal subscription is not active.)

Code Experiments (For Developers)

  • Slipmat Developer Docs (code) - public documentation for the Next Generation Slipmat
  • Datatable experiment (code)
  • @slipmatio/ui (code) - Slipmat v3 UI library (Vue 3 / TypeScript / Tailwind CSS)
